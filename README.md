# Naturetto landing page

Esta carpeta contiene la landing page de Naturetto preparada para publicarla en GitHub Pages.

Archivos principales:
- [index.html](index.html)
- [styles.css](styles.css)

Para publicar en GitHub Pages:
1. Sube esta carpeta a un repositorio de GitHub.
2. En Settings > Pages, selecciona la rama principal.
3. GitHub generará una URL pública para tu sitio.
